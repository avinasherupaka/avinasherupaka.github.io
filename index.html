Fleet Planning North America (FPNA)

The corporate Vehicle Management team is asked for an enhanced mechanism to identify and select vehicles for removal from the Rental Fleet (Activating) in order to better manage short term fleet planning in response to changes in market conditions. The Fleet Planning-North America application was written approximately 10 years ago and has undergone significant changes over its life. As a result there is a large amount of technical debt. Additionally, the size of the North American fleet has grown exponentially over time and the existing application suffers performance degradation trying to process the volume of data, which modern technologies can alleviate. Lastly, in an effort to remove dependencies from legacy applications such as Unit Services and the AS400, significant changes would need to be made to the existing application. Therefore, an application rewrite is advantageous over a redesign for Cost Going Forward.
